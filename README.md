# 🌍 Lingalo - Master Languages Professionally

Lingalo is a sleek, professional language learning platform designed for mastering Japanese, Korean, and French through a focused, dashboard-driven experience.

**🌐 Live Demo: [https://cijamie.github.io/lingalo/](https://cijamie.github.io/lingalo/)**

![Lingalo Logo](logo.png)

## 🚀 Features

-   **Multi-Language Support**: Dedicated modules for Japanese, Korean, French, and Spanish
-   **Interactive Learning Modes**:
    -   **Quiz Mode**: Test your knowledge with multiple-choice questions.
    -   **Flashcards**: Classic study method for rapid memorization.
    -   **Typing Practice**: Master the script and spelling through direct input.
    -   **Timed Challenges**: Push your speed and accuracy with 60-second sprints.
-   **Comprehensive Curriculum**:
    -   **Japanese**: Hiragana, Katakana, Dakuten, Combos, and essential vocabulary.
    -   **Korean**: Consonants, Vowels, Double Consonants, Diphthongs, and common words.
    -   **French**: Core vocabulary, travel phrases, and academic terms.
-   **Gamified Experience**: Track your streaks, earn XP, and monitor your mastery levels.
-   **Progress Persistence**: Local storage integration ensures your learning progress, mastery levels, and custom lists are saved across sessions.
-   **PWA Ready**: Install Lingalo on your mobile device or desktop for a native-app feel.
-   **Responsive Design**: A premium, "ambient-light" inspired UI that works flawlessly on mobile, tablet, and desktop.

## 🛠️ Technology Stack

-   **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3.
-   **Styling**: Custom CSS with a focus on modern glassmorphism and ambient lighting effects.
-   **Fonts**: Integrated with Google Fonts (Inter, Noto Sans JP, Noto Sans KR, Playfair Display).
-   **Persistence**: Browser `localStorage` for saving user state and progress.
-   **Offline Support**: Manifest-driven PWA configuration.

## 📱 Mobile Installation

Lingalo is built as a Progressive Web App (PWA). To install:
-   **iOS**: Open in Safari -> Share Button -> "Add to Home Screen".
-   **Android**: Open in Chrome -> Tap three dots -> "Install App".
